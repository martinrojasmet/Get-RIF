# Get-RIF

This script automates the process of performing massive queries for tax information from a specified source. By leveraging an Excel file containing the necessary details for each individual, the script efficiently retrieves the corresponding tax data and compiles the results for easy access and analysis.

**Requirements**
- pandas
- re
- playwright
- easyocr
- requests
- cv2
- numpy
- openpyxl
- notifypy
